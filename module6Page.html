The Spinning Top Maze!

Instructions: You control the blue spinning top, which can be controlled with the arrow keys.
The goal of the game is to pass through the red gate at the end of the maze.
If the end of the maze is not reached before speed hits 0, you lose.
The purple toy balls can be rolled by colliding with them, but will cause the top to slow faster.
Start the game by clicking the start button below.
Reset the maze with the reset button in the top right if stuck.
Changing the dropdown below will change what maze appears when the start button is hit.
Good Luck!
